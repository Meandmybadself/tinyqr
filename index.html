TinyQR

Turn a URL into a small QR code and print it on a Niimbot label over Bluetooth (Chrome / Edge, HTTPS).

Web Bluetooth needs a secure page and a Chromium-based browser. Safari is not supported.

URL

Prefill with ?url= or ?u= (use encodeURIComponent for the value).

Label & preview

Used if the printer is not auto-detected after connecting.

Match your roll; canvas uses head width from the model and this length along the feed. Width/height must align with printDirection (handled automatically). Model and length are remembered in this browser.

Printer

Disconnected.
